# PowerBI-Task2
Creating DashBoard using PowerBI
Q1. Load the Video Game Sales dataset, create a Donut chart that shows the 
distribution of games by genre. 
• Step 2: Create a Donut Chart 
o Select the Donut chart visual. 
o Drag 'Genre' to the Legend well. 
o Drag a unique identifier like 'Name' to the Values well and set the aggregation to Count 
(Distinct). 
Q2. Build a Gaming Market Dashboard. 
• 1. Line Chart showing the trend of sales over time by region. 
o Select the Line chart visual. 
o Drag 'Year' to the X-axis. 
o Drag 'NA_Sales', 'EU_Sales', and 'JP_Sales' to the Y-axis. 
• 2. Stacked Column Chart displaying the sales breakdown by genre and platform. 
o Select the Stacked column chart visual. 
o Drag 'Genre' to the X-axis. 
o Drag 'Global_Sales' to the Y-axis. 
o Drag 'Platform' to the Legend well.
